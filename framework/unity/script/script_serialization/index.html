Understanding hot reloading

Hot reloding

에디터가 실행되어 있는 상태에서 스크립트의 변경이 바로 적용되는것. 게임을 재시작하거나 에디터를 재시작 할 필요가 없음.

스크립트를 변경 하면 직렬화 가능한 모든 변경이 바로 적용됨.
